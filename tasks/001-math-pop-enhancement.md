# 숫자 팡팡 게임 고도화 계획

## 개요

**게임명**: 숫자 팡팡 (Math Pop)
**현재 상태**: 기본 기능 구현 완료
**목표**: 게임성 강화 및 사용자 경험 개선

---

## 현재 구현 상태

### 기능
- [x] 3단계 난이도 (쉬움/보통/어려움)
- [x] 덧셈, 뺄셈, 곱셈 문제 출제
- [x] 풍선 터뜨리기 인터랙션
- [x] 점수 시스템 (기본 10점 + 콤보 보너스)
- [x] 라이프 시스템 (3개)
- [x] 콤보 시스템
- [x] 기본 피드백 (정답/오답 표시)

### 제한사항
- ~~사운드 효과 없음~~ ✅ 개선됨
- ~~시각 효과 단순함~~ ✅ 개선됨
- 게임 모드 단일
- 파워업/아이템 없음

## 기술 요구사항

### 필요 라이브러리
- `howler.js` 또는 `use-sound`: 사운드 효과
- `canvas-confetti`: 파티클 효과
- `framer-motion`: 애니메이션 (기존 사용 중)

### 상태 관리
- 게임 설정 (사운드, 접근성) → Zustand 스토어 확장
- 파워업 상태 → 게임 내 로컬 상태

### 성능 고려사항
- 파티클 효과 최적화 (requestAnimationFrame)
- 사운드 프리로딩
- 메모리 누수 방지 (cleanup)
