> Kids + Play = Kiply  
> 놀면서 배우는 스마트 미니게임

## 📌 브랜드 개요

| 항목 | 내용 |
|------|------|
| 브랜드명 | **Kiply (키플리)** |
| 슬로건 | "놀면서 자라는 우리 아이" |
| 타겟 연령 | 6세 ~ 12세 |

## 🎯 핵심 가치

1. **쉬운 조작** - 탭/스와이프만으로 플레이 가능
2. **짧은 플레이타임** - 1회 플레이 1~3분
3. **교육적 요소** - 수학, 언어, 기억력, 반응속도 향상
4. **긍정적 피드백** - 칭찬 중심의 결과 화면
5. **안전한 환경** - 광고 없음, 인앱 결제 없음

## 🕹️ 게임 라인업

### 1. 🧮 숫자 팡팡 (Math Pop)

**컨셉**: 하늘에서 떨어지는 풍선 중 정답을 터뜨리기

**교육 요소**: 덧셈, 뺄셈, 곱셈 연습

**게임 방식**:
- 화면 상단에 문제 표시 (예: "3 + 5 = ?")
- 여러 숫자가 적힌 풍선이 떠다님
- 정답 풍선을 탭하면 점수 획득
- 오답 탭 시 생명 감소

**난이도 시스템**:
| 레벨 | 연산 | 숫자 범위 |
|------|------|----------|
| ⭐ | 덧셈 | 1~10 |
| ⭐⭐ | 덧셈/뺄셈 | 1~20 |
| ⭐⭐⭐ | 곱셈 | 구구단 |

---

### 2. 🃏 짝꿍 찾기 (Memory Match)

**컨셉**: 뒤집힌 카드에서 같은 그림 찾기

**교육 요소**: 기억력, 집중력, 패턴 인식

**게임 방식**:
- 4x3, 4x4, 5x4 그리드 선택
- 카드를 터치하여 뒤집기
- 같은 그림 2장을 연속으로 찾으면 제거
- 최소 시도 횟수로 모두 맞추기

**카드 테마**:
- 🐾 동물 (기본)
- 🍎 과일
- 🔢 숫자 + 영어 단어
- 🚗 탈것

---

### 3. 🔤 단어 퍼즐 (Word Hunt)

**컨셉**: 흩어진 글자를 조합하여 단어 완성

**교육 요소**: 한글/영어 어휘력, 철자 학습

**게임 방식**:
- 그림 힌트 제시 (예: 사과 그림)
- 흩어진 글자 버튼 표시 (ㅅ, ㅏ, ㄱ, ㅗ, ㅍ, ㅏ)
- 올바른 순서로 탭하여 "사과" 완성
- 힌트 버튼으로 첫 글자 공개 가능

**난이도**:
- 쉬움: 2~3글자 단어
- 보통: 4글자 단어
- 어려움: 영어 단어

---

### 4. 🎨 색깔 터치 (Color Touch)

**컨셉**: 지시에 맞는 색깔 빠르게 터치

**교육 요소**: 색상 인지, 반응 속도, 집중력

**게임 방식**:
- "빨간색을 터치하세요!" 음성/텍스트 지시
- 화면에 여러 색상 원이 표시
- 정답 터치 시 점수 획득
- 30초 내 최대한 많이 맞추기

**응용 모드**:
- **반대 모드**: "빨간색" 지시 → 빨간색 외 터치
- **영어 모드**: "Touch BLUE!"

---

### 5. 🐹 두더지 암산 (Mole Math)

**컨셉**: 정답 두더지만 잡기

**교육 요소**: 순발력 + 암산 능력

**게임 방식**:
- 화면에 문제 표시: "5 + 3 = ?"
- 9개 구멍에서 숫자를 든 두더지 등장
- 정답(8)을 든 두더지만 터치
- 오답 두더지 터치 시 감점

---

### 6. 🧩 순서 맞추기 (Sequence)

**컨셉**: 패턴을 기억하고 따라하기

**교육 요소**: 순차적 사고, 단기 기억력

**게임 방식**:
- 4개 색상 버튼 배치
- 시스템이 순서대로 색상 표시 (빨→파→노)
- 사용자가 같은 순서로 터치
- 라운드마다 패턴 1개씩 추가

## 🎨 UI/UX 가이드라인

### 컬러 팔레트

```
Primary:    #6C5CE7 (보라)
Secondary:  #00D2D3 (민트)
Accent:     #FF9F43 (오렌지)
Success:    #10AC84 (초록)
Background: #F8F9FA (밝은 회색)
```

### 디자인 원칙

- **큰 터치 영역**: 최소 48x48px, 권장 64x64px
- **둥근 모서리**: border-radius 16px 이상
- **명확한 피드백**: 터치 시 즉각적인 애니메이션
- **밝고 따뜻한 색상**: 파스텔 톤 + 포인트 컬러
- **큰 폰트**: 본문 18px, 제목 24px 이상

### 사운드

- 정답: 밝은 효과음 + "잘했어요!" 음성
- 오답: 부드러운 틀림 효과음 (부정적이지 않게)
- 배경음악: 밝고 경쾌한 루프

## 🏆 보상 시스템

### 점수 체계
- 정답: +10점
- 연속 정답 콤보: +5점 보너스
- 시간 보너스: 빠른 정답 시 추가 점수

### 성취 배지
| 배지 | 조건 |
|------|------|
| 🌟 첫 걸음 | 첫 게임 완료 |
| 🔥 연속왕 | 10연속 정답 |
| 🧠 천재 | 100점 이상 |
| 👑 마스터 | 모든 게임 플레이 |

### 결과 화면 메시지
- 0~30점: "좋은 시작이야! 다시 해볼까? 💪"
- 31~60점: "잘했어! 조금만 더 연습하자! 🌟"
- 61~90점: "대단해! 정말 잘하는구나! 🎉"
- 91~100점: "와! 천재인걸? 완벽해! 👑"
